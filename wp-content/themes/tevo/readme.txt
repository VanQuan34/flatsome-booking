Tevo is a premium WordPress theme designed for insurance agencies, brokers, and financial services.

== Description ==

Tevo is a modern, professional WordPress theme built specifically for insurance companies, brokers, and financial services. It features 8 custom Elementor widgets, WooCommerce integration for selling insurance products, and a fully customizable design.

**Key Features:**
* 8 Custom Elementor Widgets (Insurance Card, Pricing Table, Quote Form, etc.)
* WooCommerce Integration
* Contact Form 7 Integration
* Revolution Slider Support
* One-Click Demo Import
* WordPress Customizer with Live Preview
* 3 Header Styles (Default, Transparent, Centered)
* Fully Responsive Design
* RTL Support
* Translation Ready
* SEO Optimized
* Cross-browser Compatible

== Installation ==

1. Upload the `tevo` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Install recommended plugins when prompted
