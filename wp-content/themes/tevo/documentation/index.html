Tevo - Insurance Agency WordPress Theme

Thank you for purchasing Tevo! This documentation will guide you through the installation, setup, and customization of your theme.

Installation

Method 1: WordPress Dashboard

  1. Go to Appearance → Themes → Add New → Upload Theme
  2. Click Choose File and select tevo.zip
  3. Click Install Now, then Activate
  4. You will be prompted to install required plugins — click Begin Installing Plugins

Method 2: FTP Upload

  1. Extract tevo.zip on your computer
  2. Upload the tevo folder to /wp-content/themes/
  3. Go to Appearance → Themes and activate Tevo

One-Click Demo Import

  1. Ensure all required plugins are installed and activated
  2. Go to Appearance → Import Demo Data
  3. Select the Tevo - Insurance Agency demo
  4. Click Import and wait for completion
Important: Demo import will add content to your site. It's recommended to use on a fresh WordPress installation.

WooCommerce Setup

  1. Complete the WooCommerce setup wizard
  2. Add insurance products as WooCommerce products
  3. Configure payment gateways in WooCommerce → Settings → Payments
  4. Customize product display in Customize → Tevo Theme Options

Contact Form 7

Create your quote form in Contact → Add New, then use the shortcode in the Tevo Quote Form Elementor widget or any page.

Header & Footer

Header

Configure via Customize → Tevo Theme Options → Header. Supports 3 styles: Default, Transparent, and Centered.

Footer

Add widgets to footer columns via Appearance → Widgets. Configure columns (2-4) and copyright text in the Customizer.

Blog Settings

Configure sidebar position and excerpt length in Customize → Tevo Theme Options → Blog.

Insurance Services (CPT)

The Tevo Core plugin adds the Insurance Services custom post type:

  1. Go to Insurance Services → Add New
  2. Add title, content, featured image, and excerpt
  3. Fill in Service Details: Icon class, Starting price, Key features
  4. Assign to a Service Category

Shortcode: [tevo_services count="6" columns="3" category=""]

Translations

Tevo is fully translation-ready. Use the included .pot file in /languages/ folder with tools like Poedit or Loco Translate plugin.

Child Theme

A child theme (tevo-child) is included. Install and activate it for custom modifications. Your customizations will be preserved during theme updates.

FAQ

How do I update the theme?

Use the Envato Market plugin for automatic updates, or manually upload the new version via FTP.

Can I use Elementor Pro?

Yes! Tevo is fully compatible with Elementor Pro and supports Theme Builder locations.
